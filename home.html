Important FAQ

What is the DOM, and how is it structured?

The DOM (Document Object Model) is a hierarchical structure that represents the HTML or XML document as a tree of nodes. Each node corresponds to an element, attribute, or text in the document. The root of the tree is the document itself (document object), and from there, elements like , , and others form branches. This structure allows scripts to access, manipulate, and update the content and structure dynamically.

How do you select elements in the DOM?

You can select elements in the DOM using various methods. document.getElementById('id') selects an element by its ID, document.getElementsByClassName('class') selects elements by class name, and document.getElementsByTagName('tag') selects elements by tag name. Additionally, document.querySelector('selector') selects the first element that matches a CSS selector, while document.querySelectorAll('selector') selects all matching elements.

What is the purpose of innerHTML, and how does it differ from textContent?

innerHTML is used to get or set the HTML content of an element, allowing for dynamic changes to an element's inner structure, including HTML tags. On the other hand, textContent retrieves or sets only the text inside an element, ignoring any HTML tags. textContent is generally safer when you're only working with text to avoid security issues like Cross-Site Scripting (XSS).

How does event delegation work in the DOM, and why is it useful?

Event delegation involves attaching a single event listener to a parent element, which handles events from its child elements. Instead of adding separate event listeners to each child, the parent uses event bubbling (where events propagate up the DOM tree) to manage events efficiently. This is particularly useful for dynamically added elements, as it reduces memory usage and the need for re-adding event listeners when the DOM changes.

How can you dynamically add and remove elements from the DOM?

You can create elements using document.createElement('tagName') and then append them to the DOM using methods like appendChild() or insertBefore(). To remove elements, you can use element.remove() to directly remove an element from the DOM, or parentElement.removeChild(element) to remove it from its parent. These methods allow you to dynamically change the structure of the web page based on user interactions or other conditions.
